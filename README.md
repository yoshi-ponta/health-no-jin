# サービス概要
『ヘルスノ陣』は、家族や友人とチームを作り、日々の運動を記録しながらポイントを貯めていく健康促進サービスです。
グループごとにポイントが集計され、ポイントに応じて、自分たちの「城」が成長していきます。
応援機能や仲間の頑張りを可視化できることによってつながりを感じ、運動のモチベーションを維持・向上できるサービスとなっています。

# ⚔️サービスURL
https://health-no-jin.onrender.com

# 🧑‍🦽サービス開発の背景
介護現場で働く中で、家族や健康に対して後悔する方や「大切な人と、いつまでも元気で一緒に暮らしたい」という思いを抱く方をたくさん見てきました。このサービスを通じて、ユーザーが大切な人とのつながりを保ちつつ、楽しみながら健康を意識できるような機会を提供したいと考えています。

# 機能紹介
| ユーザー登録 / ログイン |
| :---: | 
| [![Image from Gyazo](https://i.gyazo.com/832fdbe6043a47a805d138bed0687c9b.gif)](https://gyazo.com/832fdbe6043a47a805d138bed0687c9b) |
| <p align="left">『名前』『メールアドレス』『パスワード』『確認用パスワード』を入力してユーザー登録を行います。ユーザー登録後は、自動的にログイン処理が行われるようになっており、そのまますぐにサービスを利用する事が出来ます。<br>また、Googleアカウントを用いてGoogleログインを行う事も可能です。</p> |
<br>

| グループ作成 / 参加 |
| :---: | 
| [![Image from Gyazo](https://i.gyazo.com/e5bef8778b43f6fd200155be9aed6e9c.gif)](https://gyazo.com/e5bef8778b43f6fd200155be9aed6e9c) |
| <p align="left">グループを作成すると、自動でグループのリーダーになり、大将という称号が与えられます。また、リーダーの招待リンクからグループに参加できます。<br>グループ一覧画面にて、招待リンクを送ることができます。</p> |
<br>

| 運動記録 |
| :---: | 
| [![Image from Gyazo](https://i.gyazo.com/aff37f5fb871254fc37dc088c41d8b63.gif)](https://gyazo.com/aff37f5fb871254fc37dc088c41d8b63) |
| <p align="left">運動記録画面では、各項目から簡単な操作で記録を追加できます。記録するとポイントが加算され、進捗ゲージに反映されることで、城が少しずつ成長していきます。
<br>

| 応援機能 |
| :---: | 
| [![Image from Gyazo](https://i.gyazo.com/0e1a064313be7b8f2db05414f3f3795e.gif)](https://gyazo.com/0e1a064313be7b8f2db05414f3f3795e) |
| <p align="left">グループ一覧では、メンバーごとの獲得ポイントが確認できます。一日一回まで各メンバーを応援することができます。<br>また、グループメンバーに応援されると、通知一覧に「○○さんから応援されました！」と表示されます。</p> |
<br>

| マイページ |
| :---: | 
| [![Image from Gyazo](https://i.gyazo.com/b510fcdf2edc9b3e21ff43517487e12a.gif)](https://gyazo.com/b510fcdf2edc9b3e21ff43517487e12a) |

# 技術構成

## 使用技術
| カテゴリ | 技術内容 |
| --- | --- | 
| サーバーサイド | Ruby on Rails 8.1.1・Ruby 3.3.6 |
| フロントエンド | Hotwire (Turbo / Stimulus)・JavaScript |
| CSSフレームワーク | Tailwindcss + daisyUI |
| Web API | Google OAuth (OmniAuth Google) |
| データベースサーバー | PostgreSQL |
| ファイルサーバー | AWS S3 |
| アプリケーションサーバー | Kamal (Docker)・Puma |
| バージョン管理ツール | GitHub・Git Flow |
